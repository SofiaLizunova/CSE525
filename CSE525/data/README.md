# Data Folder

This folder stores source, sample, and processed data used by the project.

## Suggested usage
- `sample/` → small example Excel or CSV files for demonstration
- `raw/` → original source data (optional, often excluded from Git)
- `processed/` → derived outputs generated by the pipeline

## Data source for this project
The project starts from an Excel-based validated dataset containing source values and intermediate calculations for the TMY-STR case study.

If the full dataset cannot be uploaded publicly, include:
- a sample file,
- or a description of the required input columns,
- and mention how the original file is loaded locally.
